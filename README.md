# AI Resume Analyzer

An AI-powered web app that analyzes resumes against job descriptions and provides improvement suggestions.

## Features
- Resume upload (PDF)
- Job description input
- Match score using NLP (TF-IDF + Cosine Similarity)
- Missing skills detection
- AI-based resume suggestions

## Tech Stack
- Python
- Streamlit
- OpenAI API
- pdfplumber
- scikit-learn

## Run Locally

pip install -r requirements.txt
streamlit run app.py
